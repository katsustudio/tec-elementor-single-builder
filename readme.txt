Elementor Single Builder for The Events Calendar Plugin.

== Description ==

You can create your desired template through this plugin and give it your own styles. This plugin works with Elementor and you must have it installed.

Using this plugin, you can create multiple templates and change your single page for all events whenever you want with just one click through the setting.

This plugin works with The Events Calendar version available in WordPress and in the near future more features will be added from the Events Calendar Pro version and its editions.
